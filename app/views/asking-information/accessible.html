What you need to tell us depends on when you completed your training:

  • before 2012
  • between 2012 and 2019
  • after 2019

This is because the accessibility guidelines have changed between these time periods.

Training before 2012

You need to tell us what type of training it was.

Training between 2012 and 2019

You need to tell us:

  • what type of training it was
  • which version of WCAG you used

Training after 2019

You need to tell us:

  • what type of training it was
  • which version of WCAG you used
  • if it was in the UK public sector

If you do not have all of this information

If you need to get more information, you can save and come back later. You can also check what is needed using the accessibility guidance on GOV.UK.
